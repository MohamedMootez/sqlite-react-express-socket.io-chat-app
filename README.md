# 💬 Chat App Backend

This is the backend of a simple real-time chat application built using:

- ⚙️ **Node.js** + **Express** for server-side logic
- 🔌 **Socket.IO** for real-time WebSocket communication
- 🗃️ **SQLite** (via `better-sqlite3`) for storing messages locally using SQL

---

## 🚀 Features

- ✅ Send messages in real time
- ✏️ Edit and delete messages
- 🗂️ All messages stored in `chat.db` using real SQL
- 🔁 New users automatically receive existing messages on join
